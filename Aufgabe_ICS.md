Erinnern Sie sich an die Wumpus World aus der ICS Vorlesung?

Ihre Aufgabe ist es, sich den Prolog-Interpreter "SWI-Prolog" zu installieren und eine Umsetzung der Wumpus-Welt darin zum Laufen zu bekommen. Anschließend sollen Sie die orthografischen Fehler im Code korrigieren und die Wumpus-Umgebung systematisch ändern.

## Vorbereitung

- Installieren Sie auf Ihrem Computer SWI-Prolog: https://www.swi-prolog.org/
- Klonen Sie sich das folgende GitHub-Repository: https://github.com/becker-asano-hdm/wumpus-prolog
- Starten Sie den Prolog Interpreter und darin dann die in der Datei "wumpus.pl" vorhandene Implementierung der Wumpus-Welt.

## Aufgaben
- Wie starten Sie das Spiel nachdem Sie es geladen haben?
- Welche Bedeutung hat ein Unterstrich '_' im Code?
- Was ist in der im Code angegebenen Welt anders als in der, die in der Vorlesung vorgestellt wurde?
- Ändern Sie die Welt so ab, dass das Gold unerreichbar wird. Was passiert dann?
